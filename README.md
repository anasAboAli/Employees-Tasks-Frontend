# 💻 الواجهة الأمامية للمنشأة (Frontend)

هذه هي الواجهة الأمامية لنظام إدارة الموارد والعمليات الموحد للشركات والمبنية باستخدام Vue 3 و TypeScript و Vite و Tailwind CSS.

## 🛠️ المتطلبات التشغيلية
- **Node.js**: إصدار 18 أو أحدث.
- **npm** أو **yarn** أو **bun** لإدارة الحزم.

## 🚀 التشغيل والتثبيت المحلي

1. **تثبيت المكتبات:**
   ```bash
   npm install
   ```

2. **تطوير واجهات المستخدم محلياً:**
   ```bash
   npm run dev
   ```
   سيبدأ خادم Vite المحلي بالعمل على المنفذ `3000`. ويقوم تلقائياً بتوجيه (Proxy) أي استدعاء للمسار `/api` إلى خادم الخلفية المستقل على `http://localhost:5000`.

3. **بناء المشروع للإنتاج (Production Build):**
   عند بناء المشروع للإنتاج للرفع على خوادم الاستضافة الثابتة (مثل Vercel, Netlify, Cloudflare Pages, S3, إلخ)، يمكنك تمرير رابط الـ Backend من خلال متغير البيئة `VITE_API_URL`:
   ```bash
   VITE_API_URL=https://your-backend-api.com npm run build
   ```
   سيتم تجميع الملفات الثابتة داخل مجلد `dist` وهي جاهزة تماماً للرفع على أي استضافة ثابتة (Static Web Hosting).

## 📂 الهيكل العام للمشروع
- `src/main.ts`: نقطة الدخول الرئيسية وتفعيل معترض الـ Fetch لربط الـ API تلقائياً.
- `src/App.vue`: الحاضن الرئيسي للواجهات وشرائط التنقل والإشعارات.
- `src/views/`: صفحات النظام الرئيسية (تسجيل الدخول، لوحة المدير العام، لوحة المشرف، لوحة الموظف).
- `src/components/`: الرسوم البيانية العربية والأدوات المساعدة المتكررة.
- `src/utils.ts`: دوال تحويل العملات وتنسيق التواريخ وإدارة الرموز والجلسات.
